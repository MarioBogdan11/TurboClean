# TurboClean

A lightweight system optimization and cleaning utility for Windows built with Python and CustomTkinter.

## Features

- **System Scan**: Analyze your system for junk files, temporary files, and optimization opportunities
- **Smart Cleanup**: Remove unnecessary files and free up disk space
- **System Boost**: Apply advanced tweaks and optimizations to improve performance
- **Programs Manager**: Review and manage installed programs

## Requirements

- Python 3.13+
- Windows OS
- Dependencies: `customtkinter`, `psutil`, `Pillow`, `pypiwin32`

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python turbocleaner.py`
